# ユーザーマニュアル — 英会話シャドウイング AI版（サブスク）

## はじめに

英会話シャドウイング AI版へようこそ。AI（Claude）が好きなテーマで英文を自動生成し、ネイティブ音声でシャドウイング練習できるアプリです。発音は AI（OpenAI Whisper）が自動採点します。

---

## アカウント登録・ログイン

### 1. アカウント登録

1. アプリのトップページで「新規登録」をタップ
2. メールアドレスとパスワードを入力
3. 登録メールに届いた認証リンクをクリック
4. 認証完了後、ログインできるようになります

### 2. サブスクリプションに加入

機能を利用するにはサブスクリプション（**月額 ¥980**）への加入が必要です。

1. ログイン後「プラン」タブから加入
2. Stripe の決済画面でクレジットカード情報を入力
3. 加入完了後、すべての機能が利用可能になります

> **更新サイクル:** ご契約日を起点に1ヶ月ごとに自動更新・自動課金されます（例: 7月17日契約なら毎月17日）。更新のたびに生成回数（45回/月）がリセットされます。

## 基本的な使い方

### 1. テーマを入力して英文を生成

1. 「テーマ」欄に練習したい話題を日本語または英語で入力
   - 例: `空港でのチェックイン`、`カフェで注文`、`仕事の打ち合わせ`
2. 「スタイル」でダイアログ（会話）またはモノローグ（パッセージ）を選択
3. 「レベル」で難易度を選択:

| レベル | 目安 |
|--------|------|
| 🌱 初級 | 中学英語・英検3級（短い文・基本単語のみ）|
| 🌿 中級 | 高校英語・英検2級（自然な日常英語）|
| 🌳 中上級 | 英検準1級・TOEIC700+（ビジネス語彙・慣用句）|
| 🏔 上級 | 英検1級・ネイティブ級（高度な語彙・複雑な構文）|

4. 「生成」ボタンをタップ
5. AIが200〜300文字の英文を生成して表示します

> **ヒント:** テーマ欄に細かい要望を書き足すこともできます（例: `空港での会話 過去形を多めに`）。

> **生成回数:** 月45回まで生成できます。回数はご契約日を起点に1ヶ月ごと（例: 7月17日契約なら毎月17日）の自動更新時にリセットされます。

### 2. 音声を再生

| ボタン | 動作 |
|--------|------|
| **▶ 再生** | 音声を再生 |
| **⏸ 一時停止** | 再生を一時停止 |
| **■ 停止** | 再生を停止 |
| **🔁 リピート** | 繰り返し再生 ON/OFF |

「速度」スライダー（0.5〜1.5倍）で再生速度を調整できます。

### 3. 音声の種類を変える

「音声」選択ボックスから好みの音声を選択:

| 音声名 | 特徴 |
|--------|------|
| **Nova** | 女性・明瞭（おすすめ）|
| **Shimmer** | 女性・柔らかめ |
| **Alloy** | 中性的 |
| **Echo** | 男性・自然 |
| **Fable** | 英国系アクセント |
| **Onyx** | 男性・低めの声 |

## 録音・採点

自分の発音を録音して、AIが自動採点します。

### 録音手順

1. 練習したい英文を生成・選択
2. 「🎤 録音」ボタンをタップ
3. マイクの使用許可が求められたら「許可」を選択
4. 音声を読み上げる
5. 「⏹ 停止」ボタンをタップ
6. 録音音声がプレーヤーで再生でき、自動的に採点が始まります

### 採点結果の見方

| スコア | 評価 |
|--------|------|
| 90点以上 | 🌟 素晴らしい！ |
| 70〜89点 | 👍 いい感じ！ |
| 50〜69点 | 💪 もう少し！ |
| 50点未満 | 📚 練習を続けよう |

> **採点の仕組み:** OpenAI Whisper（高精度音声認識）で録音音声を文字起こしし、お手本テキストと単語単位で一致度を計算しています。
>
> **対応ブラウザ:** Safari（iPhone/iPad）・Chrome（PC/Android）・Firefox・Edge すべてで採点が利用できます。

---

## フレーズ解説

「📖 フレーズ解説」ボタンで、生成された英文の **日本語訳・重要フレーズの読み・意味・使い方** を AI が表示します。

- 一度作成した解説は**テキストと一緒に保存**されます。「履歴」から再度練習するときは、AIを呼ばずに即表示されます
- 解説を作り直したい場合は、解説画面の「🔄 再生成」ボタンでいつでも更新できます（更新結果も保存されます）

---

## テキストの保存

気に入ったテキストは「📌 保存」ボタンで保存できます。「履歴」タブからいつでも呼び出して再練習できます。

## よくある質問

**Q: 1つのアカウントを複数端末で使えますか？**
A: はい。同じメールアドレス・パスワードでログインすれば、PC・スマホ問わず利用できます。

**Q: 採点が0点と表示される**
A: 音声がうまく録音できていない、またはマイクの音量が小さい可能性があります。マイクに近づいて、はっきり発音し直してください。

**Q: 「採点機能はサブスク会員限定です」と出る**
A: サブスクプランへの加入が必要です。「プラン」タブから加入してください。

**Q: 月の生成回数を超えてしまった**
A: 次回の更新日（ご契約日を起点に1ヶ月ごと。例: 7月17日契約なら8月17日）に自動リセットされます。使い切った場合は次回更新日までお待ちください。更新時にはお知らせメールが届きます。

**Q: サブスクを解約したい**
A: 「アカウント」タブの「プラン解約」から手続きできます。期間終了まで機能は利用可能です。

**Q: オフラインで使えますか？**
A: 英文生成・音声再生・採点はすべてサーバーAPIを使用するためオンラインが必要です。録音そのものはオフラインでも可能です。
